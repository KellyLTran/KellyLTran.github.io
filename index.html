RLE Program

The RLE Program leverages on Python's core capabilities and Run-Length Encoding (RLE) techniques to efficiently convert between raw image data and string formats.
The application has a significant compression ratio, reducing image sizes by up to 50% without noticeable loss in quality.
It features an intuitive, interactive GUI, crafted with the ConsoleGfx Python library, which enables users to visualize the compression process in real-time.

Binge Buddy

Binge Buddy is a movie sorting application that utilizes over 10,000 movie records from the IMDB database.
The application offers users a variety of sorting options in both ascending and descending order, such as by genre, director, and ratings.
It utilizes a timing mechanism to demonstrate the speed of the different sorting algorithms of merge sort and shell sort, where shell sort outperformed Merge Sort with an average completion time of 167,200 nanoseconds compared to Merge Sort's 1,015,900 nanoseconds
